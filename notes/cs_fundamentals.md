# Command Dictionary

# Network

A collection of computers capable of exchanging information.

#Internet

A global network connecting smaller networks together.

# Protocol

A standardized set of rules governing communication between computers.

Examples:

SSH (Secure Shell)
TCP (Transmission Control Protocol)
UDP (User Datagram Protocol)
HTTP (HyperText Transfer Protocol)
HTTPS (HyperText Transfer Protocol Secure)

# Client

Computer requesting a service.

Example:

Your laptop.

# Server

Computer providing a service.

Example:

GitHub.

# Authentication

Process of proving identity.

Examples:

Password

SSH Keys

Fingerprint

Face ID

# Public-Key Cryptography

Authentication using a mathematically linked public key and private key.

Private key:

Never shared.

Public key:

Shared freely.

# SSH

Secure Shell

Purpose:

Secure authentication and encrypted communication between computers.

# TCP

Transmission Control Protocol

Reliable communication.

Guarantees delivery.

# UDP

User Datagram Protocol

Fast communication.

Does not guarantee delivery.

# MITM Attack

Man-in-the-Middle Attack

An attacker secretly intercepts communication between two legitimate systems.

# Fingerprint

A short cryptographic identifier uniquely representing an SSH key.

# Host Verification

Process of confirming that the remote computer is genuinely the server it claims to be.

# Remote Repository

Another copy of a Git repository hosted elsewhere.

Examples:

GitHub

GitLab

Bitbucket

# Upstream

The default remote branch associated with a local branch.
