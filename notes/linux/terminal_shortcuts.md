# Linux Terminal Shortcuts & Commands

Goal:
Reduce friction.
Move faster.
Think less about tools.

---

# Terminal

## Open Terminal

Shortcut:

Ctrl + Alt + T

Use:
Open terminal instantly.

---

## Clear Terminal

Command:

clear

Shortcut:

Ctrl + L

Use:
Clean workspace.

---

## Stop Running Command

Shortcut:

Ctrl + C

Use:
Kill processes.

VERY IMPORTANT.

---

## Search Previous Commands

Shortcut:

Ctrl + R

Example:

Type:

python

Find previous commands.

One of the highest ROI shortcuts.

---

## Auto Complete

Shortcut:

Tab

Example:

cd robo[TAB]

Autocomplete.

---

## Repeat Previous Command

Shortcut:

↑

Use:
Reuse commands.

---

## Cursor Navigation

Beginning:

Ctrl + A

End:

Ctrl + E

Delete previous word:

Ctrl + W

Delete line:

Ctrl + U

---

# Folder Navigation

Current location:

pwd

List files:

ls

Detailed:

ls -la

Move:

cd

Go back:

cd ..

Go home:

cd ~

---

# Monitoring

Interactive monitor:

htop

Tree structure:

tree

Disk:

df -h

Memory:

free -h

---

# Search

Find file:

find

Search text:

grep

Example:

grep hello file.txt
