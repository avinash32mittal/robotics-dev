## Problems Encountered

### Problem 1

Downloaded VS Code .deb file but extracted instead of installing.

Cause:
Ubuntu opened archive manager.

Solution:
Installed from terminal.

Lesson:
Packages and archives are different concepts.

---

### Problem 2

formatOnSave enabled but formatting did not occur.

Cause:
No formatter selected.

Solution:
Installed Black Formatter.

Lesson:
Automation requires explicit configuration.

---

### Problem 3

Formatting worked but:

black --version

returned command not found.

Cause:
VS Code extension ≠ system installation.

Solution:
Installed Python package separately.

Lesson:
Editor environment and terminal environment behave independently.

## Engineering Observations

Machines rarely fail because of capability; they fail because assumptions remain hidden.

Today showed that software environments behave similarly.

The system only behaves as intended once dependencies, permissions, and communication paths are explicitly defined.
