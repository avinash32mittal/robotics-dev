# Coordinates

A coordinate identifies a location.

Example:

``` text
(3,5)
```

The first value is the **x-coordinate**, the second is the
**y-coordinate**.

------------------------------------------------------------------------

# Cartesian Plane

``` text
      y
      ↑
 II   |   I
------+------→ x
 III  |  IV
```

Robotics uses coordinate systems to describe robot and object positions.

------------------------------------------------------------------------

# Distance (Concept)

Distance tells how far two points are apart.

Applications:

-   Robot navigation
-   Path planning
-   Collision avoidance

------------------------------------------------------------------------

# Direction

Movement is described using direction and distance.

Examples:

-   Forward
-   Backward
-   Left
-   Right

Later this expands into vectors.

------------------------------------------------------------------------

# Measurements

Common quantities:

  Quantity   Unit
  ---------- ----------
  Length     m
  Time       s
  Speed      m/s
  Angle      ° or rad
