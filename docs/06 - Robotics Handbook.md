# What is Robotics?

Robotics combines:

-   Mechanical Engineering
-   Electronics
-   Computer Science
-   Mathematics
-   Control Engineering
-   Artificial Intelligence

to build autonomous or semi-autonomous machines.

------------------------------------------------------------------------

# What is a Robot?

A robot is a programmable machine capable of sensing, processing
information and acting on its environment.

## Robot Cycle

``` text
Sense
  ↓
Think
  ↓
Act
  ↓
Repeat
```

------------------------------------------------------------------------

# Robot Components

  Component          Purpose
  ------------------ -------------------------
  Sensors            Collect information
  Controller (CPU)   Process data
  Actuators          Produce motion
  Power System       Supply energy
  Software           Control robot behaviour

# Why Learn Programming?

Programming allows a robot to:

-   Read sensors
-   Make decisions
-   Control motors
-   Communicate
-   Perform autonomous tasks

Python is the primary language for this curriculum.
